  screen -S synth

  python pipeline/augment/synthetic.py run --species Bombus_ashtoni Bombus_sandersoni Bombus_flavidus --count 500
                                                                                                                           
  # Detach: press Ctrl+A then D                                                                                            
  
  Reconnect anytime with:                                                                                                  
                  
  screen -r synth

  Key commands:                                                                                                            
  - screen -ls — list active sessions
  - screen -r synth — reattach                                                                                             
  - Ctrl+A, D — detach (process keeps running)
  - Ctrl+A, K — kill session when done                                                                                     
                                      
  Your process survives disconnects because screen runs it in a server-side terminal that persists independently of your   
  SSH connection.
